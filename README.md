# GuardianAI: US BGV Automation for Angi Technicians

### Project Overview
This project automates the Background Verification (BGV) process for US-based technicians on the Angi platform. It uses **Generative AI** to analyze criminal records and **SQL** to track verification status.

### Key Features:
- **Automated Risk Assessment:** AI analyzes reports to find criminal history.
- **Workflow Management:** Distinguishes between 'New Entry' and 'Re-verification'.
- **Decision Support:** Provides a 'Fit for Angi' recommendation.

### Tech Stack:
- **Python** (Logic & AI Integration)
- **SQL** (Database Management)
- **Gen AI (OpenAI/Gemini API)** (Document Analysis)
